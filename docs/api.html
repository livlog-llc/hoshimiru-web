星をみるひと API

見つけた星座をシェアする「星をみるひと」の機能を利用するためのAPIです。

データ形式

戻り値の形式はJSONのみの対応となります。

星座情報の取得

観測場所から見れる星座情報を取得します。

HTTPメソッド

GET

URL

https://livlog.xyz/hoshimiru/constellation

アプリケーション登録

不要

サンプル

https://livlog.xyz/hoshimiru/constellation?lat=35.862&lng=139.645&date=2013-10-31&hour=0&min=0

パラメータ
キー値
lat観測場所の経度
lng観測場所の緯度
date観測場所の日付
hour観測場所の時
min観測場所の分
id検索絞込用、星座のID、値は星座一覧参照
disp検索絞込用、水平線より下を表示、on or offを設定
返り値
キー値
id星座番号
jpName星名（日本語）
enName星名（英語）
content内容
origin起源
eclipticalFlag楕円形フラグ
ptolemyFlagプトレマイオスフラグ
confirmed確認されている
drowing星番号一覧
directionNum方向(数値)
direction方向
altitudeNum高度(数値)
altitude高度
starImage星図
